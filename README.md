# 📚 Biblioteca Lumina

Sistema web de biblioteca desenvolvido em Python utilizando o framework Bottle e a arquitetura BMVC (Bottle Model View Controller).

---

## Nível 1 — Página web estática

Este é o primeiro nível do projeto.

O objetivo foi desenvolver uma aplicação web personalizada utilizando BMVC, HTML, CSS e JavaScript.

## Funcionalidades

- Página Inicial
- Página Sobre
- Página Acervo
- Página Contato
- Pesquisa de livros
- Filtro por categoria
- Formulário de contato
- Menu responsivo
- Logo personalizado
- Favicon
- Layout responsivo

---

## Tecnologias

- Python
- Bottle
- BMVC
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

---

## Como executar

Instale o Bottle:

```bash
python -m pip install bottle
```

Execute:

```bash
python route.py
```

Depois acesse:

```
http://localhost:8080
```
