# 📚 Biblioteca Lumina

Sistema web desenvolvido em Python utilizando Bottle e arquitetura BMVC.

## Objetivo

Este projeto está sendo desenvolvido em quatro níveis conforme a Macro Tarefa BMVC.

## Evolução

- ✅ Nível 1 — Site institucional da biblioteca
- 🚧 Nível 2 — CRUD de livros
- ⏳ Nível 3 — Login e área restrita
- ⏳ Nível 4 — Comunicação em tempo real com WebSocket

## Tecnologias

- Python
- Bottle
- HTML5
- CSS3
- JavaScript
- Bootstrap
